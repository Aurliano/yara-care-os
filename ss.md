# پیشنهاد من

## Android Hub

```text
Min SDK = 29 (Android 10)

Target SDK = آخرین نسخه پایدار (فعلاً 36)

Compile SDK = آخرین نسخه پایدار (فعلاً 36)
```

چرا؟

* Android 10 پایه خوبی برای Device Owner و Kiosk است.
* BLE و سرویس‌های پس‌زمینه بالغ‌تر شده‌اند.
* بیشتر تبلت‌های جدید تجاری حداقل Android 10 یا 11 دارند.
* اگر خودت سخت‌افزار را انتخاب می‌کنی، دلیلی ندارد خودت را درگیر Android 8 یا 9 کنی.

# یک پیشنهاد که شاید مهم‌ترین تصمیم سخت‌افزاری پروژه باشد

من اصلاً اجازه نمی‌دهم مشتری هر تبلتی بخرد.

من یک لیست رسمی منتشر می‌کنم:

```
Supported Devices

✔ Samsung Tab A9

✔ Lenovo Tab M11

✔ Xiaomi Redmi Pad SE

...
```

و فقط این‌ها را تست و پشتیبانی می‌کنیم.

این کار سه مزیت بزرگ دارد:

1. هزینه پشتیبانی به‌شدت کاهش پیدا می‌کند.
2. Kiosk Mode و Device Owner روی دستگاه‌های تست‌شده قابل اعتمادتر خواهند بود.
3. تجربه کاربری برای همه مشتریان یکسان می‌شود.
